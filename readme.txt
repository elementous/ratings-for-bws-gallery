This plugin makes Rating Manager plugin fully compatible with BWS Gallery plugin. Adds image, post and comment rating form.

== Description ==

This plugin is made to add full compatibility between Rating Manager and BWS gallery plugins. Now you can easily add rating form to any BWS gallery and it allows your website visitors to rate every image from any gallery. Every separate gallery page also can be rated. All rating results will be available on statistics page for more convenient way to analyze the rating data for each image and each gallery.

Rating Manager is the most advanced WordPress content and comment rating plugin. It utilizes SVG image technology and it allows you to modify rating form icons easily without any graphical editor. Basically Rating Manager plugin offers unlimited rating form icon variation count, from classical star icon to any integrated SVG icon or SVG icon uploaded by you. Rich snippets for enhanced SEO features, comment rating, comprehensive statistics, feedback form and more functions and features available. Compatible with any theme right out of the box.

= Features =

* Enables ratings for images on BWS galleries.
* Enables ratings for gallery pages.
* Enables rating statistics for images on BWS galleries and BWS gallery pages.

= Requirements =

* Rating Manager plugin (version 1.0.2 or higher).
* BWS gallery plugin (version 4.3.8 or higher).

== Installation ==

1. Install and activate Rating Manager and BWS gallery (you can skip this step if you have Rating Manager and BWS gallery plugins already installed).
2. Upload `ratings-for-bws-gallery` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does it work stand-alone? =
No. This plugin requires Rating Manager plugin and BWS Gallery plugin to be installed.

= Can I use it with other WordPress gallery plugins? =
No. This plugin is tailored for BWS gallery plugin. Rating Manager is capable to deal with all posts, pages, custom posts generated by other plugins and media files, but in some cases it need some minor modifications. This plugin adds necessary modifications for full compatibility between Rating Manager and BWS gallery plugins.

= Can I change appearance of rating form? =
Yes. Rating Manager allows you to change rating form icons, colors and size of icons right from the settings page. There is no need to use any software for image editing because it uses SVG vector images.
